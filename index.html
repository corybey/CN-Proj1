Project Objective

This project builds a publicly accessible website and demonstrates how networking concepts such as DNS resolution, IP addressing, protocols (HTTP/HTTPS), and security apply in real deployments.

Key statement: Our website is hosted publicly, DNS resolves the domain to an IP address, and communication occurs over HTTPS using TCP/IP.
