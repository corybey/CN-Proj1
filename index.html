Project Objective

This project enhances a public website by adding security, database integration, and performance optimization. The website demonstrates how networking concepts such as DNS, IP addressing, HTTPS, and secure client-side rendering apply in real-world deployments.

Implemented features: HTTPS, XSS prevention through sanitized rendering, Firestore database integration, and front-end performance optimization.
